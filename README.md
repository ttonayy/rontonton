# RonTonTon Quiz
Bu proje, Ronya ve Tonay için özel olarak hazırlanmış bir quiz oyunudur. 
İstanbul'daki tarihi bir yer hakkında bir soru içerir. 
Doğru cevabı verdiğinizde ekran renk değiştirerek kutlama yapar!
## Kullanım
1. Sayfayı açın.
2. Sorunun cevabını kutuya yazın.
3. "Gönder" butonuna tıklayın.
## Teknolojiler
- HTML
- CSS
- JavaScript
